# AI Spend Audit Dashboard

A modern AI-powered expense tracking dashboard built with Next.js, Tailwind CSS, TypeScript, and Recharts.

## Features

- Expense Tracking
- Local Storage Persistence
- Spending Analytics
- Pie Chart Visualization
- Search & Category Filters
- AI Spending Insights
- Responsive Dashboard UI

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Recharts

## Installation

```bash
npm install
npm run dev
```
